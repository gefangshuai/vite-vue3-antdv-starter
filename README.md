Vue3 starter Base on Vite and Ant Design Vue
----
# Usage
```bash
git close git@github.com:gefangshuai/vite-vue3-antdv-starter.git
cd vite-vue3-antdv-starter
yarn install
yarn dev
```

# Dependencies
- axios
- ant-design-vue
- mitt
- lodash
- dayjs
- vue-router
- vuex
- [frontend-utils](https://github.com/gefangshuai/frontend-utils)
